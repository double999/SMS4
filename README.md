# SM4 Sbox 塔域实现
本项目完成了国密算法SM4中S盒的硬件实现
实现方法为同构同构映射矩阵，将求逆运算最终转化到GF(2)上进行，比直接用查找表实现节省面积
适合面积限制型的应用
